Overview

EtherWAN and AMASTek's Motor Predictive Diagnostic Maintenance utilizes IoT techniques with Azure IoT service like reliability-centered maintenance, overall equipment effectiveness, and analytics to significantly improve maintenance efficiency, asset availability, and lifetime value. Limited visibility on asset status and performance, high costs of managing aging legacy assets, high asset replacement costs and other overhead contribute to overall maintenance spending. Our solution helps practitioners evaluate assets and diagnose problems using monitoring tools like sensors. We leverage advanced algorithms and machine learning principles to deploy repeatable maintenance solutions and an optimized asset lifecycle maintenance plan.

MPDM Solution by EtherWAN x AMASTek

Key Features:

  • Provides full records and remote monitoring with persistent data collection and storage
  • Aids in the timely replacement of failing parts and continuous operations using resonance analysis, torque & abnormal temperature monitoring, unbalanced detection and any gear damage detection
  • Real-time remote asset visibility and control enabled
  • Notifies personnel of minor to severe system anomalies with automated alerts to gain faster maintenance
  • Intelligence in the cloud and at the edge
  • Integrated Microsoft Azure IoT service and SaaS

The Product Offering:
In this MPDM solution, the intelligence edge computing leveraging by EtherWAN AiR PACE IoT Smart edge computing gateway seamlessly extends Azure IoT to edge devices so they can act locally on the data they generate, while still using the cloud for management, analytics, and durable storage.
