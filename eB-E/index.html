Overview

EtherWAN industrial cellular gateway EW200 has now been Microsoft Azure certified for IoT and is now listed on Microsoft Azure IoT Device Catalog. This indicates that the IoT Gateway is certified and tested to be compatible with Microsoft's cloud-based services and that the device works with Azure IoT technology. By choosing an Azure IoT certified device, customers can save time and simplify integration of their device with cloud software solutions for whichever IoT application they need.

EtherWAN's first Azure certified device EW200 series is an intelligent industrial grade gateway that combines IEEE 802.11 a/b/g/n/ac WLAN and 4G LTE cellular technologies designed for critical infrastructure and industrial applications. It offers flexible connectivity over cellular and Wireless LAN for harsh environments, giving operations and IT real-time awareness with remote monitoring. It allows up to 4 Ethernet device links and provides dual SIM cards for failover redundancy to ensure uninterrupted connectivity. Moreover, it delivers best-in-class connectivity with multi path – wireless, Ethernet, serial, I/O interfaces satisfy most applications Global LTE options and make it easy to get connected.

Features
  • Global HSPA+ and 4G LTE support and certified for the world's major carrier networks
  • Integrated Modbus, MQTT & OPC-UA server for advanced IoT applications
  • Equipped with a dual-SIM LTE module with failover function
  • Wi-Fi Compliance IEEE 802.11 a/b/g/n/ac 2T2R; Multiple (8) SSIDs for privilege group devices or users
  • 4 x Gigabit Ethernet ports with tag based and port based VLANs for easy control of groups and traffic
  • VPN tunnel auto-reconnect when the cellular network has unexpected interruptions in service
  • Data logging off-line proxy function which automatically executes pre-defined Modbus ASCII/RTU data acquisition task when Modbus TCP Query Timeout
  • Open Python programming development platform for developers
  • Seamless power failover between wide-range redundant DC 12V-48V power inputs
